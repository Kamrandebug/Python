Q1 — Positive or Negative
Ask the user to enter a number. Print whether it is positive, negative, or zero.

Q2 — Voting Eligibility
Ask the user their age. Print whether they are eligible to vote or not.

Q3 — Biggest Number
Ask the user to enter two numbers. Print which one is bigger. If both are equal, print "Both are equal."

Q4 — Grade Checker
Ask the user to enter their marks out of 100 and print their grade:

80 and above → A
60 to 79 → B
40 to 59 → C
Below 40 → Fail

Q5 — Even or Odd
Ask the user to enter a number. Print whether it is even or odd.
(Hint: use the % operator)

Q6 — Login System
Store a username and password in your code. Ask the user to enter both. If both match, print "Login Successful"; otherwise, print "Wrong credentials."

Q7 — Temperature Checker
Ask the user to enter the temperature. Print:

Above 40 → "Very Hot"
25 to 40 → "Normal"
10 to 24 → "Cold"
Below 10 → "Very Cold"

Q8 — Leap Year
Ask the user to enter a year. Print whether it is a leap year or not.
(Hint: A year is leap if it is divisible by 4)

Q9 — Discount System
Ask the user to enter their total shopping bill. Apply discount:

Above Rs. 5000 → 20% discount
Above Rs. 2000 → 10% discount
Otherwise → No discount

Print the final amount after discount.

Q10 — Triangle Valid or Not
Ask the user to enter three angles of a triangle. If they add up to 180, print "Valid Triangle"; otherwise print "Invalid Triangle."

Q11 — Weekday or Weekend
Ask the user to enter a day number (1–7). Print whether it is a weekday or weekend.
(1–5 weekday, 6–7 weekend)

Q12 — BMI Calculator
Ask the user to enter their weight (kg) and height (meters). Calculate BMI and print the result:

Below 18.5 → "Underweight"
18.5 to 24.9 → "Normal"
25 and above → "Overweight"

(Hint: BMI = weight / height²)
